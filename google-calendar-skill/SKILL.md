# Google Calendar Skill

Google Calendar API integration for CrewX. Provides full CRUD operations for calendar events.

## Setup

### Initial Setup

```bash
# 1. Install the skill template
crewx template init google-calendar-skill

# 2. Navigate to the skill directory
cd google-calendar-skill

# 3. Install dependencies
npm install

# 4. Get Google OAuth Credentials (see below)
# 5. Run authentication
node setup-auth.js
# This opens a browser for Google login and saves credentials to .env
```

> **Note**: The skill can be installed anywhere. All commands below assume you are in the skill directory.

### Google Cloud Console Setup

1. Go to [Google Cloud Console](https://console.cloud.google.com/)
2. Create or select a project
3. Enable "Google Calendar API"
4. Create OAuth 2.0 credentials (Desktop app type)
5. Download `credentials.json` to the skill directory

## Commands

All commands should be run from within the skill directory.

### List Calendars
```bash
node list-calendars.js
```

Shows all calendars connected to your Google account with their IDs.

### List Events
```bash
# Primary calendar, next 7 days
node list-events.js

# Specific calendar
node list-events.js your@email.com

# Custom time range
node list-events.js --days 14
node list-events.js primary --days 30
```

### Create Event
```bash
# Basic event
node create-event.js "Meeting" "2025-12-05T10:00:00" "2025-12-05T11:00:00"

# With location
node create-event.js "Lunch" "2025-12-05T12:00:00" "2025-12-05T13:00:00" --location "Restaurant"

# With attendees
node create-event.js "Review" "2025-12-05T14:00:00" "2025-12-05T15:00:00" \
  --attendees "alice@example.com,bob@example.com"

# With description
node create-event.js "Weekly Sync" "2025-12-05T09:00:00" "2025-12-05T09:30:00" \
  --description "Weekly team sync meeting"

# Different calendar
node create-event.js "Personal" "2025-12-05T18:00:00" "2025-12-05T19:00:00" \
  --calendar your@email.com
```

### Update Event
```bash
# Update title
node update-event.js <event-id> --summary "New Title"

# Update time
node update-event.js <event-id> --start "2025-12-05T15:00:00" --end "2025-12-05T16:00:00"

# Update location and description
node update-event.js <event-id> --location "New Place" --description "Updated notes"
```

### Delete Event
```bash
node delete-event.js <event-id>

# From specific calendar
node delete-event.js <event-id> --calendar your@email.com
```

## Environment Variables

Stored in `.env` in the skill directory (auto-generated by setup-auth.js):

| Variable | Description |
|----------|-------------|
| `CALENDAR_CLIENT_ID` | Google OAuth Client ID |
| `CALENDAR_CLIENT_SECRET` | Google OAuth Client Secret |
| `CALENDAR_REFRESH_TOKEN` | OAuth Refresh Token |

## Files

| File | Description |
|------|-------------|
| `setup-auth.js` | OAuth authentication setup |
| `calendar-client.js` | Shared Calendar API client |
| `list-calendars.js` | List all calendars |
| `list-events.js` | List calendar events |
| `create-event.js` | Create new event |
| `update-event.js` | Update existing event |
| `delete-event.js` | Delete event |

## CrewX Integration

This skill can be used by AI agents:

```yaml
# In crewx.yaml
skills:
  - calendar
```

Agent can then:
- List user's calendars to find the right one
- Check upcoming events for scheduling
- Create new events with attendees
