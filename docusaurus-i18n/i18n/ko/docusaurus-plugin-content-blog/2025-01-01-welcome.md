## 소개

이 블로그는 프로젝트의 최신 소식, 튜토리얼, 그리고 기술 정보를 공유하는 곳입니다.

## 자동 번역 워크플로우

이 사이트는 **CrewX 자동 번역 워크플로우**를 사용합니다:

1. **한글 블로그 작성** - 이 파일처럼 `i18n/ko/docusaurus-plugin-content-blog/`에 작성
2. **번역 확인** - `npm run translate:check`로 미번역 파일 확인
3. **자동 번역** - `npm run translate:ko-to-en`으로 영어 번역 자동 생성
