# 새로운 문서 사이트에 오신 것을 환영합니다!

축하합니다! CrewX를 사용한 AI 기반 번역 워크플로우가 포함된 Docusaurus 사이트를 성공적으로 설정했습니다.

<!--truncate-->

## 🎉 무엇이 포함되어 있나요?

이 템플릿에는 다국어 문서 작성을 시작하는 데 필요한 모든 것이 포함되어 있습니다:

### 📝 AI 기반 자동 번역
- 한국어로 블로그를 작성하면 자동으로 영어로 번역
- Claude AI를 사용하는 CrewX 에이전트 기반
- 의미와 문맥을 보존하는 자연스럽고 전문적인 번역

### 🤖 3개의 AI 에이전트가 준비되어 있습니다

1. **@blog_manager** - 콘텐츠 작성 도우미
   - 블로그 아이디어 브레인스토밍
   - 완전한 블로그 글 작성
   - 기존 콘텐츠 편집 및 개선

2. **@blog_translator** - 전문 번역가
   - 한국어 ↔ 영어 번역
   - 코드 블록과 포맷 보존
   - 기술 블로그 톤

3. **@template_installer** - 설정 가이드
   - 구성 도움말
   - 템플릿에 대한 질문 답변
   - 문제 해결

## 🚀 빠른 시작

### 첫 블로그 작성하기

```bash
# 블로그 매니저를 사용해서 한국어 블로그 작성
crewx x "@blog_manager 시작하기에 대한 한국어 블로그를 작성해줘"

# 자동으로 영어로 번역
npm run translate:ko-to-en
```

### 또는 Slack 봇 사용하기

```bash
# Slack 인증 정보 입력
vim .env.slack

# 봇 시작
./start-slack.sh
```

Slack에서 봇을 멘션하면 됩니다:
```
@CrewX 새로운 기능에 대한 블로그를 작성해줘
```

## 📚 다음 단계

1. **사이트 커스터마이징**
   - `docusaurus.config.ts`에서 프로젝트 정보 업데이트
   - `static/img/`에 로고 추가
   - `src/css/custom.css`에서 색상 커스터마이징

2. **콘텐츠 작성**
   - `i18n/ko/docusaurus-plugin-content-blog/`에서 블로그 작성
   - `npm run translate:ko-to-en`으로 자동 번역
   - 또는 `blog/`에서 영어로 직접 작성

3. **배포**
   - 빌드: `npm run build`
   - Vercel, Netlify 또는 GitHub Pages에 배포

## 💡 팁

- 한국어 블로그 파일은 `i18n/ko/docusaurus-plugin-content-blog/`에 작성
- 영어 번역본은 `blog/`로 자동 저장
- 두 언어 모두 동일한 파일명 사용
- 번역 스크립트가 자동으로 미번역 파일 감지

## 🤝 도움이 필요하신가요?

템플릿 설치 도우미 에이전트에게 물어보세요:
```bash
crewx q "@template_installer 홈페이지를 커스터마이징하는 방법을 알려줘"
```

즐거운 문서 작성 되세요! 🎊
